# Site web KléIA Solutions

Site commercial de l'activité de formation et conseil IA d'Anne-Cécile Le Dain, sous la marque KléIA Solutions. HTML/CSS/JS pur, sans framework, hébergé sur GitHub Pages.

**Domaine :** kleia-solutions.fr (Infomaniak, CNAME configuré)

## Pages

| Fichier | Page |
|---|---|
| `index.html` | Accueil (hero, preuve, profils, offres, quiz maturité IA, cas d'usage, CTA) |
| `services.html` | Services détaillés + FAQ (JSON-LD) |
| `formations.html` | Catalogue des formations intra-entreprise + tarifs |
| `quiz-maturite-ia.html` | Lead magnet : quiz « Score de maturité IA », 10 questions, capture email |
| `cas-usage.html` | Cas d'usage filtrables par métier |
| `temoignages.html` | Témoignages clients (carrousel + page dédiée) |
| `a-propos.html` | Parcours et positionnement d'Anne-Cécile (JSON-LD Person) |
| `contact.html` | Formulaire Formspree + lien de réservation Google Calendar |
| `mentions-legales.html` | Mentions légales |
| `confidentialite.html` | Politique de confidentialité (RGPD) |
| `blog/index.html`, `blog/*.html` | Articles de blog SEO/GEO |
| `assets/css/styles.css` | Charte KléIA + composants |
| `assets/js/main.js` | Nav, carrousels, filtres, animations |
| `sitemap.xml`, `robots.txt`, `CNAME` | SEO technique |

## Mise à jour du site

Le dossier source de vérité est `PROJECTS/formatrice-ia/site-kleia-v3/` (hors de ce repo, dans les documents de travail). Toute modification doit être faite là-bas, testée en local, puis recopiée dans ce repo avant de pousser.

## Prise de rendez-vous et contact

- Lien de réservation : Google Calendar Appointment Schedule, synchronisé avec le calendrier personnel pour éviter les conflits.
- Formulaire de contact : Formspree, avec un filtre Infomaniak Mail pour recevoir les notifications en boîte de réception principale.
